# Classic Games

8 classic HTML5 games built from scratch with vanilla JavaScript. No frameworks, no dependencies, no ads. Fully responsive for mobile & desktop.

## Games

| Game | Description | Controls |
|---|---|---|
| [Tetris](https://korifcan.github.io/Classic-Games/games/tetris/) | Classic block stacking puzzle | ← → ↓ rotate, touch swipe |
| [Snake](https://korifcan.github.io/Classic-Games/games/snake/) | Eat food, grow, don't crash | Arrow keys, touch swipe |
| [2048](https://korifcan.github.io/Classic-Games/games/2048/) | Merge tiles to reach 2048 | Arrow keys, touch swipe |
| [Flappy Bird](https://korifcan.github.io/Classic-Games/games/flappy/) | Tap to fly through pipes | Space/click, tap |
| [Space Shooter](https://korifcan.github.io/Classic-Games/games/space/) | Blast aliens & bosses | Arrows + space, drag |
| [Tic Tac Toe](https://korifcan.github.io/Classic-Games/games/ttt/) | Play against AI (Minimax) | Click/tap cells |
| [Memory](https://korifcan.github.io/Classic-Games/games/memory/) | Match pairs of cards | Click/tap cards |
| [Minesweeper](https://korifcan.github.io/Classic-Games/games/mine/) | Find all mines | Left/right click, tap/long-press |

## Features

- **Zero dependencies** — pure HTML, CSS, JavaScript
- **Touch support** — all games work on mobile
- **Keyboard support** — all games work on desktop
- **Score tracking** — high scores saved in localStorage
- **Dark theme** — neon/glow aesthetic
- **Self-contained** — each game is a single HTML file
- **No ads** — nothing to distract
